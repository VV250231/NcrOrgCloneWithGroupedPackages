Usage :
- sm: optional, if want to have small version
- options: required, NG scope reference for list of item to show.
- model: required, NG scope model binding reference.
- value-key: optional, NG-scope model property name, which value will be used as Label.
- disabled: optional, if disabled the control is required.
- inverse: optional, to enable variation on gray background.

<div
    class="di-dropdown"
    model="<NG-SCOPE-MODEL>"
    options="<NG-SCOPE-LIST>"
    inverse="{true | false}"
    sm="{true | false}"
    disabled="{true | false}"
    value-key="<NG-SCOPE-MODEL property name>"
    ></div>
