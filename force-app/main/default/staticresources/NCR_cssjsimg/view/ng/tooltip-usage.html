Tool Tip \ Pop-over (on click)

 
Usage :
- icon: optional, Font icon name excluding Bootstrap 'glyphicon-' prefix.
- position: default top, where to display tooltip.
- content: body message for tooltip.
<span class="di-tooltip" inverse="{true | false}" icon="<icon-name>"
    position="left | right | bottom | top" content="{text | HTML | NG scope reference}">

Tool Tip \ Pop-over (variation on gray background)

 

 
 
Usage :
- icon: optional, Font icon name excluding Bootstrap 'glyphicon-' prefix.
- inverse: optional, enable variation on tooltip over gray.
- position: default top, where to display tooltip.
- content: body message for tooltip.
<span class="di-tooltip" inverse="{true | false}" icon="<icon-name>"
                        position="left | right | bottom | top" content="{text | HTML | NG scope reference}">
